Django Rest API

As part of my web development journey, I worked on a Django REST API project that involved developing and implementing CRUD operations using Python and generic API views. The project aimed to create a secure and efficient API that would allow users to access data and perform various operations. To enhance the search functionality of the API, I integrated Algolia search engine, providing users with faster and accurate results. I also utilized SimpleJWT, Javascript web client, and DjangoCors to implement a secure login system with token authorization, ensuring that users had secure access to the API. Throughout the project, I utilized various technologies such as Django REST Framework, Python, Javascript, Algolia, SimpleJWT, and DjangoCorsHeaders.
